Mi Página Web

La World Wide Web fue creada por Sir Tim Berners-Lee en 1989. Berners-Lee es un ingeniero y científico de la computación británico, y su objetivo era permitir el acceso a la información a través de la Internet.

Lenguaje HTML

HTML es un lenguaje de marcado que permite crear estructuras de página web. HTML es el acrónimo de Hypertext Markup Language (Lenguaje de marcado de hipertexto), y se utiliza para crear elementos de página web, como títulos, imágenes, enlaces, entre otros.

Áreas de interés

Mis áreas de interés dentro del programa educativo son: programación web, inteligencia artificial, y seguridad informática.
